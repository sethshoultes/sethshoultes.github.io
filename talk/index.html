Talk to Seth.

A live conversation with a HeyGen LiveAvatar of Seth, voiced by an ElevenLabs clone of his actual voice. Click start, allow microphone access, and ask anything. The avatar listens, thinks, and answers in real time.

Click "Start the call" to begin

The session opens a microphone connection and streams an avatar of Seth back. Each call uses real LiveAvatar credits, so it'll close itself after a few minutes of silence.

What it is
A real-time avatar driven by HeyGen LiveAvatar. The avatar's likeness is Seth; the voice is an ElevenLabs clone of Seth's voice.
How it works
A small Cloudflare Worker mints a one-shot session token. The page opens a WebRTC connection to LiveAvatar with that token. The API key never leaves the server.
Privacy
Audio streams direct to LiveAvatar / ElevenLabs; nothing is logged here. Sessions auto-close on silence and on tab close.
